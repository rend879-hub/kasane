# ChatGPTレビュー依頼テンプレート

以下はClaudeによるKASANEの実装報告です。

あなたはKASANEのPM兼レビュー担当として確認してください。

## 見る観点

1. 要件定義とのズレ
2. UI/UXの違和感
3. 実装上のリスク
4. 文言・世界観のズレ
5. 次にClaudeへ投げるべき差分修正内容

## 条件

- 大改修ではなく差分修正を優先する
- KASANEの現方針から外れた提案はしない
- 初期段階では機能を増やしすぎない
- 文言・世界観のズレを重視する
- 最後にClaudeへそのまま投げられるMarkdownプロンプトを出す

## 前提として守ること

- `/docs/dev-rules.md` の方針に従う
- 1日1枚のシェアカード生成器に集中する
- ログイン、SNS、コメント、ライブラリ機能を初期実装しない
