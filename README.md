# KASANE Docs Starter

KASANE開発用の運用テンプレート一式です。

## 使い方

このZIPを解凍し、`docs` フォルダをKASANEのGitHubリポジトリ直下に配置してください。

配置後の例：

```text
kasane/
├─ app/
├─ components/
├─ public/
├─ docs/
│  ├─ dev-rules.md
│  ├─ claude-report-template.md
│  └─ chatgpt-review-template.md
├─ README.md
└─ package.json
```

## Claudeへの基本指示

```md
/docs/dev-rules.md を前提に作業してください。
実装後は必ず /docs/claude-report-template.md の形式で報告してください。
```
